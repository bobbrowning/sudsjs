getInstruction 

   GET INSTRUCTION

Turns a search specification into an sql search and bindings Won't work with MONGO

getInstruction(table: string, spec: Object): array
Parameters
table (string) Table name
spec (Object) Filter specification - see above
Returns
array: instruction + bindings
Example
Typical  filter specification
{
 search: {                                             
   andor: 'and',
   searches: [
      ['userType', 'eq', 'C'],
      ['fullName', 'contains', 'Acme']  
    ]
  },
  sort: ['id', 'DESC'],
  limit: 20,
}

createTable

Create new tables on the database according to the schema. Only new tables are added. It's purpose is to do the heavy lifting of setting up the database. Fine tuning you need to use a different tool but make sure it is in syncwith the schema.

Parameters
req (object)
